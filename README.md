# Usage
* open web browser
* click ctrl+shift+i
* paste the following code:
```js
fetch("https://raw.githubusercontent.com/larix45/quizizz-fetcher/main/ver2.js")
.then((res) => res.text()
.then((t) => eval(t)))
```
* enjoy :)
